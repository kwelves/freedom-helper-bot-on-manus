# 🕊 Freedom Assistant — Бот-хелпер для АН «Свобода»

Тихий помощник группы анонимных наркоманов «Свобода». 6-й игрок команды.

## Возможности

- **Напоминания о собраниях** — автоматически по расписанию (Пн, Чт в 19:00, Сб в 17:00)
- **Теги участников** — в раскрываемой цитате (уведомления приходят, но визуально не мешают)
- **Приветствие новичков** — тёплое сообщение при входе нового участника
- **Автосбор юзернеймов** — бот запоминает всех, кто пишет или заходит в группу
- **Управление списком** — добавление/удаление юзернеймов через команды

## Команды (только для владельца)

| Команда | Описание |
|---------|----------|
| `/add @username` | Добавить юзернейм в список тегов |
| `/remove @username` | Удалить юзернейм из списка |
| `/list` | Показать текущий список участников |
| `/test` | Отправить тестовое напоминание в группу |
| `/help` | Показать справку |

Также бот реагирует на слово **«хелпер»** в чате (только от владельца).

## Деплой на Railway

### 1. Создай репозиторий на GitHub

1. Зайди на github.com и создай новый репозиторий (например `freedom-bot`)
2. Загрузи туда все файлы из этой папки

### 2. Подключи к Railway

1. Зайди на [railway.app](https://railway.app) и войди через GitHub
2. Нажми **New Project** → **Deploy from GitHub repo**
3. Выбери свой репозиторий `freedom-bot`
4. Railway автоматически определит Procfile и начнёт деплой

### 3. Настрой переменные окружения

В Railway зайди в настройки сервиса → **Variables** и добавь:

```
BOT_TOKEN=8615058016:AAH2f7NLEsbrtKGMGdNxATPXky3fsHFSmCc
OWNER_ID=8142241137
CHAT_ID=-1003764364716
```

### 4. Готово!

Бот запустится автоматически и будет работать 24/7.

## Файлы

- `bot.py` — основной код бота
- `requirements.txt` — зависимости Python
- `Procfile` — конфигурация запуска для Railway
- `runtime.txt` — версия Python
- `usernames.json` — список юзернеймов для тегов (обновляется автоматически)
